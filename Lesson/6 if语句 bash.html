bash

基本语法

  1. if <TEST>; then
  3. else
  5. fi

<TEST> 可以是一个 [ ], [[ ]], (( )), 也可以是进程输出结果

条件是单中括号表达式[ ]

数字比较

  1. a=1
  2. [ $a -lt 0 ]
  1. if [ $a -lt 0 ];then echo right;else echo wrong; fi
  1. if [ $a -lt 0 ]; then
  2.    echo right
  3. else 
  4.    echo wrong
  5. fi

数字比较,先将字符串转化成数字再比较

  1. b="122"
  2. a=123
  3. let b=b+1
  5. [ $a -eq $b ] && echo hi
  6. [ $a = $b ] && echo hi
  7. [ $a -lt "123" ] && echo hi

条件是双中括号表达式[[ ]]

也就是字符串比较
About Matching Of String
=~

  1. [[ "123" =~ 123* ]]
  2. [[ "123" =~ [0-9]+ ]]
  3. [[ "123" =~ [0-9]{0} ]]

比较> =

  1. [[ "123" > 1* ]] && echo hi
  2. [[ "123" = "123"* ]] && echo hi

双中括号里数字与字符串的比较区别

  1. [[ "123" < 23 ]] && echo hi
  2. [[ 123 < 23 ]] && echo hi

比较[]

  1. [ "123" = 123 ]

Others:

  1. ls /tmp
  2. [[ -e /tmp || -e /NOT_EXIST ]] && echo hi
  3. [ -e /tmp -o -e /NOT_EXIST ] && echo hi
  5. Same with `-a` and `&&`
  7. [[ -e /tmp && -e a.log ]] && echo hi

字符串比较

  1. [[ 3 > 19 ]] && echo hi 
  2. [ "3" \> "19" ] && echo hi
  1. [[ A > a ]] && echo hi 
  2. [[ A < b ]] && echo hi #bourne定义 从小到大：0-9aA-zZ

单括号的字符串比较，第一个大于即可。

  1. [ a21 \> A210 ] && echo hi

双括号的字符串比较，第一个大于,或其余字符都相等时，任意一个大于另一个即可。

  1. [[ A21 > a21 ]] && echo hi
  2. [[ A21 > a210 ]] && echo hi #错
  4. [[ A21 < A12 ]] && echo h
  6. [[ A21 > a2* ]] && echo hi # *==space
  8. [[ a > 10000000* ]] && echo hi
比较定义 从小到大 编码
bouren定义 由小到大 0123456789aAbBcCdD...zZ
ASCII 由小到大 0123456789A-Za-z

条件是双小括号表达式(( ))

For math operation

  1. a=1
  2. let a=a+1
  3. (( a=a+1 ))
  4. echo $? #0
  5. (( a=a+1, a==4 ))
  6. echo $?

  1.  if cat -n a.log | head -n 3 && [[ -e /NOT_EXIST ]];then 
  2.  echo hi;
  3.  else echo bi; 
  4.  fi
