#bash的数组

数组要用$来访问
```
a=(seq 3)
echo ${a[@]} #seq 3
```
区别 
```bash
a=`seq 3` #a是一个变量
echo $a

a=$(seq 3)
```
```
a=( `seq 3` )
echo ${a[0]} #run a and show the first result
```
## ${a[@]}
```
echo ${a[@]}
echo ${a[8

echo ${a[1]}
echo ${a[3]}
```
## ${#a[@]}
```
echo ${#a[@]} #how many vectors are there in a 
```

#Vector vs Array
```
a=`seq 10`
echo $a
echo ${a:1:5}

a="12345678910}
echo ${a:0:3}
```
