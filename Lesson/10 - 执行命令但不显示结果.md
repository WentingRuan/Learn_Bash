# 10 : 执行命令但不显示结果

标签（空格分隔）： bash

---

# :=
如果一个向量有值，不执行赋值命令；如果没有值，则赋值，但不返回结果
```
echo $b
: ${b:=3}
echo $b

: ${b:=123}
echo $b

```
```
a="echo hi"
$a # $ == excecute
echo $a
```
```
unset b
c=${b:=123}
$c
echo $c
$b
echo $b
```
# :-
```
unset $b
: ${b:-123}
echo $b
```

# ：表示执行命令但是不输出结果
```
b=123
: ${b:-789}
echo $b
```

```
work(){ :; }
: ${b:=3}
: 789
```
