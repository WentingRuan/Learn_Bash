10 : 执行命令但不显示结果

bash

:=

如果一个向量有值，不执行赋值命令；如果没有值，则赋值，但不返回结果

  1. echo $b
  2. : ${b:=3}
  3. echo $b
  5. : ${b:=123}
  6. echo $b
  1. a="echo hi"
  2. $a # $ == excecute
  3. echo $a
  1. unset b
  2. c=${b:=123}
  3. $c
  4. echo $c
  5. $b
  6. echo $b

:-

  1. unset $b
  2. : ${b:-123}
  3. echo $b

：表示执行命令但是不输出结果

  1. b=123
  2. : ${b:-789}
  3. echo $b
  1. work(){ :; }
  2. : ${b:=3}
  3. : 789
